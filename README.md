## English Version

### Overview

This project implements a **physics-informed wideband impedance modeling framework** for induction motors, covering frequencies up to **10â€“100 MHz**, a range typically underrepresented in conventional equivalent circuit models.

The approach combines analytical modeling, parasitic estimation, and optimization-based parameter fitting to achieve accurate magnitude and phase prediction while maintaining low model complexity.

It enables direct analysis of:

- CM (Common-Mode) impedance  
- DM (Differential-Mode) impedance  
- High-frequency EMI/EMC behavior in inverterâ€“motor systems
